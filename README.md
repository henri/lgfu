About
--------

This is an open source (GNU GPL v3 or later) project which is designed to provide you with a list of groups a user belongs to on OS X system(s).

License: [GNU GPL v3 or later][1]


Usage Instructions
---------

chmod +x ./lgfu.rb
./lgfu.rb myusernme
